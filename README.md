# Diablo RPG — Homebrew de Mesa

**Versão:** 1.0 (Playtest)  
**Autor:** Paulo Souza

> Sistema de RPG de mesa inspirado na franquia **Diablo**, com turnos rápidos, decisões claras e uma dificuldade que emula o modo Hardcore.

## 📖 Acesse o Livro

**https://felipe1072-git.github.io/diablo-shadowdark-rpg/**

O livro completo está online — Livro do Jogador, 18 classes, Livro do Mestre, Bestiário e Glossário.

## Princípios do Jogo

- **Velocidade:** Turnos rápidos, sem longas deliberações
- **Desafio:** Emula a sensação de jogar Diablo no modo Hardcore
- **Imersão:** Atmosfera sombria e visceral da franquia Diablo

## Conteúdo

| Seção | Descrição |
|---|---|
| Livro do Jogador | Regras, combate, magia, arsenal, origens e talentos |
| Classes | 18 classes jogáveis (Nv 1–10) |
| Livro do Mestre | Encontros, tesouros, calibração e registro de campanha |
| Criaturas | Bestiário completo com statblocks por bioma |
| Glossário | Todos os termos e condições do sistema |
| Cenários | 8 módulos de aventura prontos para jogar |

## Estrutura do Repositório

```
repo/
├── docs/                    ← fonte do site (MkDocs)
│   ├── jogador/             ← Livro do Jogador
│   ├── classes/             ← 18 classes
│   ├── mestre/              ← Livro do Mestre
│   ├── cenarios/            ← 8 módulos de aventura
│   └── assets/              ← imagens e CSS
├── livro-do-jogador.md      ← arquivo fonte completo
├── cap3-classes.md          ← arquivo fonte completo
├── livro-do-mestre.md       ← arquivo fonte completo
├── apendice-criaturas.md
├── apendice-glossario.md
└── mkdocs.yml               ← configuração do site
```
