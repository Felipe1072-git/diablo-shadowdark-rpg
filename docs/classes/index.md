### **Guia de Leitura: As Classes de Santuário**

Cada classe neste sistema foi desenhada para representar um arquétipo clássico de Diablo, adaptado para mecânicas de sistemas *d20/Shadowdark*. Ao consultar uma página de classe, observe os seguintes elementos:

* **Dado de Vida (DV):** Define a robustez física da classe. Representa os Pontos de Vida ganhos por nível.  
* **Mana (Recurso):** O valor base de mana, calculado como (10 \+ Modificador de Atributo Primário \+ Nível). É o combustível para habilidades especiais.  
* **Armaduras Permitidas:** Categorizadas pelo peso (Leve, Média, Pesada). Classes com restrição a "Leve" não podem usar armaduras metálicas complexas, priorizando mobilidade.  
* **Armas Permitidas:** Categorizadas pelo peso (Leve, Média, Pesada). Determina quais tipos de armamento o personagem é treinado para utilizar com eficiência.  
* **Atributo Principal:** O pilar mecânico da classe. Define o sucesso em testes de habilidade e o poder das suas técnicas.  
* **Habilidades de Classe:** As características únicas que definem o estilo de jogo, sejam elas técnicas de combate, magias ou talentos passivos.

### O Atributo Primário

O atributo primário de cada classe controla quatro coisas ao mesmo tempo:

| Função | Fórmula |
| :---- | :---- |
| **Mana máximo** | 10 + Mod. Primário + Nível |
| **DC de resistência** | 10 + Mod. Primário |
| **Testes dos lacaios** | Mod. Primário (ATK e testes) |
| **Mecânicas exclusivas** | Varia por classe (CA alternativa, limite de summons, etc.) |

> Classes com dois atributos (ex: FOR ou CON) escolhem um na criação. O escolhido define Mana, DC e testes de lacaios.

### Regras Universais de Classe

**DC de Resistência:** Sempre que uma habilidade forçar um inimigo a fazer um teste de resistência, a dificuldade é **DC = 10 + Modificador do Atributo Primário** da classe. Essa fórmula se aplica a todas as classes salvo indicação contrária explícita no texto da habilidade.

**Escalonamento de Dano:** Habilidades com dano escalável seguem o padrão: **dado base + 1 dado extra por 2 Mana adicional**. O jogador decide quantos dados quer rolar antes de gastar a Mana. Exemplo: uma habilidade com base 1d6 pode gastar +2 Mana para 2d6, +4 Mana para 3d6, e assim por diante.

**Talento Duplicado:** Se ao subir de nível você rolar um talento que já possui, pode escolher livremente qualquer outro talento da lista da sua classe que ainda não possua (equivalente ao resultado 20).
